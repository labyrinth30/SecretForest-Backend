### 프로젝트 설치 및 실행
* node.js와 docker가 설치되어 있어야 합니다.
1. 이 레포지토리를 클론합니다.
2. `cd` 로 프로젝트 폴더로 이동합니다.
3. 'docker-compose up --build' 명령어를 실행하여 프로젝트를 빌드하고 실행합니다.
4. `http://localhost:3000`으로 접속하여 프로젝트를 확인합니다.(true가 반환됩니다.)
